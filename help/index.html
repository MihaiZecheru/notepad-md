Introduction

Notepad MD is a simple, lightweight markdown editor where your documents, which can be written or uploaded, are safely stored using Google's Realtime Database. While markdown is a universal text format, Notepad MD's markdown is slightly different than what you might be used to. An abundance of shortcuts and tools make it easy to write and edit your documents using the supported markdown listed below. When you're done writing, you can export or print the document's markdown and compiled HTML to a variety of formats, including PDF, HTML, plain text. The HTML can be copied to your clipboard so you can paste your document elsewhere, for example a Canvas assignment or an important email. This text editor is designed to be used on a desktop, laptop, or tablet, meaning it's not recommended to use this editor on a mobile device.

After creating your account here, you will have full access to Notepad MD and all of it's features. To make an account, go to https://notes.mzecheru.com/account/me/ and click "New Document". You will then be redirected to the main page of the website where you will be spending most of your time: https://notes.mzecheru.com/document/edit.

The markdown you write in the right textbox will be compiled into HTML and displayed on the left side of the screen when the "Save" button is clicked or the Ctrl+S shortcut is used. All documents can be viewed in the Documents tab, where you can edit, view, delete, export, or share any of your documents to other users, although they'll only be able to view it without editing.

Export Options

Exporting your document is easy. You can export your document by printing it or as a PDF, HTML, NMD, or Markdown file. HTML and Markdown exports can be either saved as a file or copied to your clipboard.
Your document can be shared with others by sending them a link using the "Share Document" option. Those who have the link can view your document, but they cannot edit it unless you set them as an author from the settings menu.

Print Document: print the document or save it as a pdf.
Make A Copy: create a copy of the document for yourself.
Copy HTML: copy the HTML of the document to your clipboard using either linked or inline styles.
Copy Markdown: copy the Markdown of the document to your clipboard.
Download .html: export the document as an HTML file.
Download .nmd: export the document as an NMD file. NMD files are essentially the raw HTML from the document without any of the Notepad MD stylings.
Download MD: export the document as a Markdown file.

Settings

Title: The document's title. Give your document a memorable name.
Description: The document's description. What's your document about? Write a short summary!
Visibility: The document's visibility specifies who can see your document. Public: anyone with the link can view. Private: only you can view.
Type: The document's type changes the writing mode. Markdown is default; the text will compile into HTML. Plain Text disables compiling; leaves the document as is. Code turns on syntax highlighting, disables autocorrect, and disables compiling.
Language: The document's language determines syntax highlighting is the document Type is Code, otherwise it's an indicator and disables autocorrect if not set to English.
Theme: The document's theme determines the color and background of the document when type is Code.
Font: The document's font. Choose something pretty!
Font Size: The document's font size in pixels, which can be between 5 and 50. The default is 16.
Tab Size: The amount of spaces that the tab key will add, can be between 2 and 10. The default is 8.
Authors: The collaborators / writers of the document. Who has permission to make changes to your document? To give a user edit permissions to your document, type their email below and press enter. To revoke their permission, press the X below.
